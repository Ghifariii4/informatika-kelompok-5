Konfigurasi Mikrotik

Internet Gateway Routing

Pada penerapannya kami menggunakan hardware Mikrotik tipe hEX lite.

Tujuan konfigurasi adalah mengatur lalu lintas jaringan agar data dari satu perangkat bisa mengalir ke jaringan lain melalui Mikrotik.
Mikrotik berperan sebagai router yang membagi akses internet ke klien-klien yang terhubung.

Konfigurasi PC Client

  1. Setting IP address ether1 & ether2

    ip address add address=192.168.54.1/24 interface=ether2 ip address add address=192.168.0.74/24 interface=ether1

  2. Cek IP address

    ip address print

  3. Setting IP DNS

    ip dns set allow-remote-requests=yes servers=8.8.8.8,8.8.4.4

  4. Cek IP DNS

    ip dns print

  5. Setting IP Route

    ip route add gateway=192.168.0.1 distance=1

  6. Cek IP Route

    ip route print
